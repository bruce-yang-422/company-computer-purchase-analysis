詳細評分標準說明

CPU評分標準

評分方式：基於標準跑分結果（Cinebench R23多核性能）進行評分，避免因品牌不同、核心計算方式不同而失真。

評分公式：強度 = (Cinebench R23多核分數 / 最高分數) × 10，上限10分。

參考跑分範圍：

  • • AMD R5 8600G: 12,000-13,000分 → 7.2/10
  • • Intel Core Ultra 5 225: 15,000-16,000分 → 7.5/10
  • • Intel Core Ultra 5 245K: 22,000-24,000分 → 8.8/10
  • • AMD R7 9700X: 28,000-30,000分 → 9.2/10
  • • Intel Core Ultra 7 265K: 32,000-35,000分 → 9.8/10

註：實際跑分可能因測試環境而異，建議參考最新評測數據。

RAM加權計分標準

評分方式：基礎分數 + 品牌加權 + 功能加權，避免僅看容量而忽略品牌、速度、功能差異。

計分公式：

  • • 基礎分數：32GB = 8.0分，64GB = 9.0分
  • • 品牌加權：金士頓 +0.5，威剛 +0.3
  • • RGB加權：+0.2
  • • A/I雙模加權：+0.3（註：A/I = AMD/Intel，同時支援EXPO/XMP，此AI非彼AI）

價格趨勢：記憶體價格在2025年第四季度大幅上漲，從$2,700漲至$10,300，漲幅達281%！CP值從★★★★★降至★☆☆☆☆。同規格（32GB DDR5-5600 CL36）但價格差異巨大。

主機板加權計分標準

評分方式：使用MB100標準化分數 + 擴充性與功能加成，避免僅看MB100而忽略擴充性、功能差異。

MB100標準化公式：MB_1to10 = 1 + 9×(MB100/100)，5分為一般標準。

MB100等級對照：

  • • Prime系列 = 50(MB100) → 5.5分
  • • TUF Gaming = 60(MB100) → 6.4分
  • • AORUS Elite（低段）= 65(MB100) → 6.85分

擴充性與功能加成：

  • • 尺寸加權：ATX +0.8，M-ATX +0
  • • 供電相數加權：12相以上 +0.2
  • • WiFi版本加權：WiFi7 +0.3，WiFi6E +0.2
  • • 保固加權：5年保 +0.2

價格時間趨勢標準化換算說明

所有零件價格已標準化換算，避免空值過多，便於比較價格趨勢：

  • • 記憶體：換算成每32GB價格（64GB÷2，32GB直接使用）
  • • SSD：換算成每1TB價格（2TB÷2，1TB直接使用，500GB×2）
  • • CPU：換算成每1000分Cinebench R23多核性能的價格（基於實際跑分，避免Intel/AMD核心架構差異失真）
  • • 主機板：按MB100=60標準化（標準化價格 = 實際價格 × (60 / MB100)）
