# 公司電腦購買紀錄分析

這是一個公司電腦購買紀錄的視覺化分析網頁，使用 GitHub Pages 部署。透過數據分析與視覺化圖表，深入分析公司電腦採購的成本效益、價格趨勢與配置策略。

## 📊 專案概述

本專案記錄並分析了公司在 2025年6月至2026年1月期間的電腦採購情況，包含：
- **總採購金額**：$367,808
- **總訂單數**：7 筆
- **總電腦數**：12 台
- **平均單價**：$30,651

## ✨ 主要功能

### 1. 總覽統計
- 總訂單數、總金額、平均單價、總電腦數
- 電腦類型分布（文書機、工作開發機、伺服器）

### 2. 各電腦詳細分析
- 每台電腦的完整配置清單
- 零件價格占比分析
- 分配人員資訊

### 3. 零件價格分析與 CP 值評估
- **CPU 價格與 CP 值排名**：基於 Cinebench R23 多核性能評分
- **記憶體價格趨勢**：追蹤價格變化，發現 2025年Q4 漲幅達 281%
- **主機板價格與 CP 值**：考慮擴充性、保固、功能等因素

### 4. 成本效益分析
- 各電腦類型的成本結構分析
- CPU、主機板、記憶體、儲存等零件占比

### 5. 時間趨勢分析
- 同等級零件價格時間趨勢圖表
- 標準化換算（記憶體/32GB、SSD/1TB、CPU/1000分性能）

### 6. 綜合六邊形指標分析
- 評估各電腦在**效能、穩定、安全、管理、擴充、成本**六個面向的表現
- 雷達圖視覺化比較

## 🔍 關鍵發現

### 價格趨勢
- **記憶體價格暴漲**：從 2025年7月的 $2,700 漲至 2026年1月的 $10,300，漲幅達 **281%**
- 記憶體佔比從早期 13.4% 提升至後期 29.2%，成為成本最大變數
- CPU、SSD、主機板價格相對穩定，變化幅度在 10-20% 以內

### 配置策略
- **文書機**：早期配置（$20,210）性價比最高，後期因記憶體漲價成本增加至 $33,550-$35,284
- **工作開發機**：採用 Intel Core Ultra 7 265K + 64GB RAM + Gen5 SSD，配置最完整
- **伺服器**：以儲存為重點，NAS硬碟佔 41.7%，使用 Linux Ubuntu 24 LTS

### CP 值建議
- **最佳購買時機**：2025年7月（記憶體 $2,700，CP值★★★★★）
- **CPU 推薦**：AMD R5 8600G 性價比最高，Intel Core Ultra 5 245K 多工處理佳
- **主機板推薦**：AMD 平台 TUF B650M-PLUS ($4,790) 是性價比最高的 TUF 選擇

## 📝 評分標準

### CPU 評分
基於 Cinebench R23 多核性能跑分，避免因品牌、核心架構差異而失真。

### 記憶體評分
基礎分數 + 品牌加權 + 功能加權（RGB、A/I雙模等）。

### 主機板評分
使用 MB100 標準化分數 + 擴充性與功能加成（ATX/M-ATX、WiFi版本、保固等）。

詳細評分標準請參考分析頁面底部的說明。
